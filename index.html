Tristan Espinoza

I'm a creative coder living in Los Angeles. I use code as creative material to negotiate the joys and frustrations of understanding each other, and mostly focus on developing interactive applications that invite participants to connect and overlap in weird, spontaneous ways. My work situates technology in sociopolitical contexts and is informed by human relationships, emotional vulnerability, and infrastructure legibility.

This page will be here as I spend time navigating life things and updating the rest of the website! In the meantime I'll keep links to recent work and documentation here.

Work

A Ray, Tracing Between You and I

A Ray, Tracing is a project I made for the show Late Bloomers at Supplyframe DesignLab in Pasadena, CA. It uses computer vision techniques to extract new contours from a scene and overlay the shapes on top of a live feed.

This was made in C++ with openFrameworks.

TEDxPasadena

This project was used at the TEDxPasadena event in September 2018. The organizers wanted to personalize attendees' badges in a way that promoted conversation and collaboration without the transactional assessments associated with typical badges that listed an attendee's name and professional / institutional affiliations.

This application took a word or phrase from a participant and used it to dynamically create geometric art, which was then milled onto a sticker that could be placed on the attendee's badge.

I used C++ and openFrameworks for this project.

The Girl Who Sees

I spent a good chunk of the year collaboratively developing The Girl Who Sees, a video game about Filipinx history and mythology. I was responsible for the "adventure" aspect of the game, working on narrative flow, transitions, inventory, etc.

The game is being built in Unity. The scripting I did was primarily in C#, but I learned some basic Lua to accommodate a plug-in they were using for narrative actions called Fungus.

Dwayne Detector

The Dwayne Detector is a command-line application that uses an image or video feed to analyze the faces in the input and assess whether or not the face belongs to Dwayne "The Rock" Johnson.

The Dwayne Detector was built with Python using dlib's Python bindings.
